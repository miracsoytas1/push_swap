## Description
push_swap is a constrained sorting challenge that takes a list of unique integers, uses two stacks, and outputs a sequence of stack operations that sorts the list in ascending order with as few operations as possible.

## Instructions
- Build: `make`
- Run: `./push_swap 2 1 3 6 5 8`
- Clean: `make clean` or `make fclean`
